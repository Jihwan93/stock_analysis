삼성전자(005930) 주가 변동성 분석 프로젝트  
2025-01-01 ~ 2025-11-26까지의 삼성전자 주가 데이터를 활용하여  
일간 수익률, 변동성, 주요 이벤트 영향을 분석하는 프로젝트입니다.


1. 프로젝트 개요
이 프로젝트는 삼성전자 주가의 변동성을 분석하고  
특정 날짜의 급등·급락 원인을 뉴스 이벤트와 연계하여 분석하는 것이 목표입니다.
 


2. 데이터 설명
데이터 출처: 직접 수집  
기간: 2025-01-01 ~ 2025-11-26
사용한 변수:
Open : 시초가  
Close : 종가  
High : 고가  
Low : 저가  
Volume : 거래량  


3. 분석 과정 

3-1. 데이터 전처리  
- Date 컬럼을 datetime으로 변환  
- 인덱스를 날짜 기준으로 정렬  
- 결측치 제거  

3-2. 일간 수익률(Daily Return) 계산  
- 종가 기준 수익률 계산  
- 최대·최소 수익률 날짜 탐지  

3-3. 이벤트 분석
